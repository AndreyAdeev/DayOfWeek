Определение дня недели по дате рождения (версия 1.0)

- запустиете файл DayOfWeek.jar (без использования консоли)
- введите в текстовое поле день(число) в формате 00 (ДД);
- введите в текстовое поле месяц(число) в формате 00 (ММ);
- введите в текстовое поле год(число) в формате 0000 (ГГГГ);
- нажмите кнопку "Нажмите, для получения результата";
- в текстовом поле появится день недели, в который вы родились;
